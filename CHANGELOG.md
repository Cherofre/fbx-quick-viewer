# 更新日志

## v1.0.3

- 新增拖入单个 `.fbx` 文件直接临时预览能力。
- 拖入 FBX 后会在左侧顶部显示临时项，同时保留已经打开的文件夹列表。
- 拖入贴图仍沿用原有 UV 贴图预览逻辑。
- 当当前列表超过 300 个模型时，按滚动位置懒加载缩略图，避免几千上万个模型导致长时间卡顿。
- 新增最小回归测试，覆盖拖拽路由、缩略图保护和版本号。

## v1.0.0

首个正式发布版本，包含以下能力：

- 支持扫描文件夹并递归读取 FBX 文件
- 支持列表和缩略图两种浏览模式
- 支持缩略图生成与本地缓存
- 支持收藏、搜索、排序和历史目录
- 支持基础模型统计信息查看
- 支持标准材质、线框、顶点色、UV 预览等显示模式
- 支持法线显示、双面显示、地面网格和相机控制
- 提供中文 README 与本地打包发布配置
