# 更新日志

## v1.0.5

- 安装版支持在后台自动下载后续版本，右下角显示百分比、已下载大小和速度，下载过程不阻塞模型浏览与操作。
- 下载完成后可选择“重启安装”，更新程序会静默覆盖当前安装目录并重新启动应用。
- 手动运行新版安装包时会从注册表读取旧版安装目录，自动覆盖原位置，无需重新选择文件夹。
- 更新时保留 AppData 数据，并在覆盖安装前临时保护安装目录旁的 `FBX_Data`，安装完成后原位恢复。
- 便携版继续使用发布页手动下载，避免自动覆盖正在运行的便携程序。
- 支持从列表或缩略图中把 FBX 模型作为真实文件拖到 3ds Max 等外部程序。
- 拖放图标缩小为 32×32，避免 Windows 原生拖放预览遮挡目标窗口。
- 文件右键菜单支持查找并复制同名 `.mesh` 路径，同目录优先，也会逐级检查上级目录。
- UV 数值拖动降低灵敏度，支持 `Shift` 精调、`Ctrl` 加速和双击单项重置。
- UV 平铺值现在可以正确设置为 `0`，并支持 `Ctrl+Z` 撤销、`Ctrl+Y` / `Ctrl+Shift+Z` 重做。
- 修复更换预览贴图时仍访问旧版 UV 控件导致重置异常的问题。
- 修复 UV 控件配置声明晚于应用启动，导致动画渲染循环和鼠标视角控制未启动的问题。
- 拖出操作统一通过受控 preload API，并只允许当前窗口拖出存在的 `.fbx` 文件。
- 补充拖出功能回归测试和 3ds Max 使用限制说明。
- Electron 升级到 43.1.0，electron-builder 升级到 26.15.3，清除已知 npm 依赖漏洞。
- 新增 `?` / `F1` 中文操作速查、模型悬浮拖拽提示和“⋯”操作入口。
- 当前模型路径栏新增 `.mesh` 复制按钮，路径操作统一显示成功或错误提示。
- 首次目录加载和首次 UV 预览提供可关闭的一次性提示，并在本地记住关闭状态。
- 新增首次设置，可选择 `.mesh` 操作、视口操作模式、默认资源视图和操作帮助。
- 优化缩略图操作布局，收藏位于右上，“⋯”位于左下；路径栏使用紧凑 `M` 按钮。
- 搜索框移动到排序栏下方。
- 3ds Max 模式支持中键拖动过程中按下或松开 `Alt`，即时切换平移和旋转。
- 检查到新版本时显示 GitHub Release 更新说明。
- U/V 数值区域新增常驻拖动标识和仅在非默认值时出现的单项重置按钮。
- 降低搜索框聚焦强度，并修复点击画布后仍保持聚焦的问题。

## v1.0.4

- 新增检查更新功能，可手动检查 GitHub Release 最新版本，并低频自动检查。
- 新增多 UV 通道检测，UV 工具箱只显示当前模型实际存在的 UV 通道。
- UV 展开图和贴图预览支持跟随当前 UV 通道切换。
- 模型统计区显示当前模型 UV 通道信息，已解析模型可显示 UV2 标识。
- 文件夹扫描改为异步批处理，并显示扫描进度和取消按钮。
- Electron 渲染层迁移到 preload + contextIsolation，并通过受控本地文件协议加载 FBX/贴图。
- 缩略图缓存优先放在软件目录旁的 `FBX_Data/fbx_cache`，避免默认占用 C 盘 userData。
- 打包白名单只包含运行时必要文件，减少发布包冗余内容。
- 新增 v1.0.4 功能回归测试。

## v1.0.3

- 新增拖入单个 `.fbx` 文件直接临时预览能力。
- 拖入 FBX 后会在左侧顶部显示临时项，同时保留已经打开的文件夹列表。
- 拖入贴图仍沿用原有 UV 贴图预览逻辑。
- 当当前列表超过 300 个模型时，按滚动位置懒加载缩略图，避免几千上万个模型导致长时间卡顿。
- 新增最小回归测试，覆盖拖拽路由、缩略图保护和版本号。

## v1.0.0

首个正式发布版本，包含以下能力：

- 支持扫描文件夹并递归读取 FBX 文件
- 支持列表和缩略图两种浏览模式
- 支持缩略图生成与本地缓存
- 支持收藏、搜索、排序和历史目录
- 支持基础模型统计信息查看
- 支持标准材质、线框、顶点色、UV 预览等显示模式
- 支持法线显示、双面显示、地面网格和相机控制
- 提供中文 README 与本地打包发布配置
