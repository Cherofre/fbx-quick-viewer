# FBX 快速预览器 v1.0.0

首个正式发布版本。

## 版本简介

FBX 快速预览器是一个基于 Electron + Three.js 的本地 FBX 模型查看工具，面向需要快速浏览、筛选和检查 `.fbx` 文件的使用场景。

## 主要功能

- 支持扫描文件夹并递归读取 FBX 文件
- 支持列表视图与缩略图视图切换
- 支持自动生成并缓存缩略图
- 支持收藏、搜索、排序和历史目录
- 支持模型顶点数、三角面数等基础统计信息
- 支持标准材质、线框、顶点色、UV 预览等显示模式
- 支持法线显示、双面显示、地面网格和视角切换
- 支持 DDS、TGA 及常见图片贴图预览

## 发布产物

- 安装版：`fbx-quick-viewer Setup 1.0.0.exe`
- 便携版：`fbx-quick-viewer 1.0.0.exe`

## 适用环境

- Windows x64

## 使用方式

1. 启动程序
2. 点击“打开文件夹”
3. 选择包含 `.fbx` 文件的目录
4. 在左侧列表或缩略图中选择模型进行预览

## 说明

- 程序会在运行目录附近生成 `FBX_Data` 用于保存缓存、收藏和窗口状态
- 首次进入目录时，缩略图会逐步生成，属于正常现象
