# FBX 快速预览器

一个基于 Electron + Three.js 的本地 FBX 模型快速预览工具，适合在 Windows 环境下快速浏览、筛选和检查 `.fbx` 文件。

## 功能特点

- 扫描文件夹并递归查找 FBX 文件
- 支持拖入单个 FBX 文件直接临时预览
- 支持从列表或缩略图把 FBX 作为真实文件拖入 3ds Max 等外部程序
- 右键模型可复制同名 `.mesh` 路径，支持 `.mesh` 位于 FBX 同目录或上级目录
- 列表视图与缩略图视图切换
- 自动生成并缓存缩略图
- 大目录下按滚动位置懒加载缩略图，避免长时间卡顿
- 支持收藏、搜索、排序与历史目录
- 支持模型基础统计信息查看
- 支持标准材质、线框、顶点色、UV 预览等显示模式
- 支持当前模型多套 UV 通道检测、展开图切换与贴图预览
- 贴图预览参数支持精细拖动、单项重置和撤销/重做
- 提供 `?` / `F1` 中文操作速查、悬浮操作入口和一次性上下文提示
- 首次启动可设置 `.mesh` 操作、3ds Max 操作模式、默认列表/缩略图视图和操作帮助
- 支持法线显示、双面显示、地面网格与视角控制
- 支持 DDS、TGA 及常见图片贴图预览
- 安装版支持低频检查并在后台自动下载 GitHub Release 新版本，显示进度且不阻塞当前操作
- 便携版保留手动打开发布页下载的更新方式

## 运行环境

- Windows 10 / 11
- Node.js 22.12 及以上（仅源码开发和打包需要）
- npm

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm start
```

## 打包发布

生成可执行目录：

```bash
npm run pack
```

生成安装包和便携版：

```bash
npm run dist
```

## 使用说明

1. 启动程序后点击“打开文件夹”。
2. 选择包含 `.fbx` 文件的目录。
3. 在左侧列表或缩略图中选择模型。
4. 在右侧查看模型，并使用显示模式、相机工具和 UV 工具辅助检查。
5. 也可以把单个 `.fbx` 文件直接拖入预览区域临时查看；原文件夹列表会保留，拖入文件会显示在左侧顶部。
6. 从左侧列表或缩略图按住模型拖到 3ds Max 视口，可像从资源管理器拖入文件一样交给 Max 导入。
7. 悬停模型后点击“⋯”或右键模型，选择“复制 .mesh 路径”；程序会先检查 FBX 同目录，再逐级检查上级目录中的同名 `.mesh` 文件。当前模型也可使用路径栏右侧的 `.mesh` 按钮。
8. 当目录里模型很多时，缩略图会随着滚动动态生成；也可以手动选择生成前 300 个或生成全部。
9. 如果模型存在多套 UV，可在 UV 工具箱里切换 UV 通道查看展开图和贴图预览。
10. 点击左上角更新按钮可手动检查新版本。

按 `F1` 或点击目录栏右侧的 `?` 可随时打开中文操作速查。

3ds Max 操作模式下，中键拖动用于平移；拖动过程中按下 `Alt` 会直接切换为旋转，松开 `Alt` 可继续平移。

发现新版本时，更新窗口会显示 GitHub Release 中的更新说明，并提供“打开发布页”按钮。

安装版检测到新版本后会在后台下载，右下角显示进度、大小和速度；下载完成后可选择重启并静默覆盖原安装目录。手动运行新版安装包也会自动识别旧版目录。更新会保留 AppData 数据和安装目录旁的 `FBX_Data`。便携版不会自动覆盖自身。

UV 数值操作支持：拖动带 `↔` 的 U/V 标签调整，按住 `Shift` 精调，按住 `Ctrl` 加速；数值偏离默认值时可点击 `↶`，也可双击标签重置单项；`Ctrl+Z` 撤销，`Ctrl+Y` 或 `Ctrl+Shift+Z` 重做。

> 3ds Max 是否显示 FBX Import 对话框由 Max 和 FBX Importer 决定。如果 Max 以管理员身份运行，Windows 可能阻止普通权限的本程序向其拖放文件。

## 数据与缓存

程序会在运行目录附近创建 `FBX_Data` 文件夹，用于保存：

- 收藏数据
- 缩略图缓存
- UV 元数据缓存
- 窗口状态
