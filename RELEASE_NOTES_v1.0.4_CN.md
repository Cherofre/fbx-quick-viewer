# FBX 快速预览器 v1.0.4 发布说明

## 新增

- 支持检查更新，可手动检查 GitHub Release 最新版本。
- 启动后会低频静默检查更新，发现新版本时提示打开下载页。
- 支持检测当前模型实际存在的多套 UV 通道。
- UV 工具箱新增 UV 通道选择器，只显示当前模型存在的通道。
- UV 展开图和拖入贴图预览支持跟随当前 UV 通道切换。
- 模型统计区显示 UV 通道信息，已解析模型可显示 `UV2` 或 `UV2*` 标识。

## 注意

- 检查更新只打开 Release 页面，不自动下载或覆盖安装。
- UV 通道标识只在模型被加载或解析后显示，不会为了打标而批量解析整个目录。
