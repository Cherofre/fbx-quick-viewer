# FBX 快速预览器 v1.0.4 发布说明

## 新增

- 支持检查更新，可手动检查 GitHub Release 最新版本。
- 启动后会低频静默检查更新，发现新版本时提示打开下载页。
- 支持检测当前模型实际存在的多套 UV 通道。
- UV 工具箱新增 UV 通道选择器，只显示当前模型存在的通道。
- UV 展开图和拖入贴图预览支持跟随当前 UV 通道切换。
- 模型统计区显示 UV 通道信息，已解析模型可显示 `UV2` 或 `UV2*` 标识。
- 文件夹扫描改为异步批处理，并在侧栏显示扫描进度和取消按钮，降低超大目录扫描时的卡顿风险。
- 新增受控本地文件加载通道，在启用 Electron 安全隔离后仍可加载本地 FBX 与贴图资源。

## 优化与修复

- 缩略图缓存不再默认写入 C 盘 userData；portable 版和安装版会优先使用软件目录旁的 `FBX_Data/fbx_cache`。
- 如果安装目录不可写，会提示用户选择缩略图缓存目录，并记住选择。
- 安装包只打入运行时必要文件，不再夹带测试、协作规则或运行期缓存目录。
- 新增 Electron smoke 测试，覆盖 preload API 和本地文件协议读取。

## 注意事项

- 检查更新只打开 Release 页面，不自动下载或覆盖安装。
- UV 通道标识只在模型被加载或解析后显示，不会为了打标而批量解析整个目录。
- 如果安装目录没有写入权限，首次生成缩略图时需要选择一个缓存目录。
