# FBX 快速预览器 v1.0.5 发布说明

## 新增

- 安装版支持后台自动下载后续更新，右下角会显示下载百分比、大小和速度，下载期间可以继续浏览和操作模型。
- 下载完成后可点击“重启安装”，程序会静默覆盖当前安装目录并重新启动。
- 手动运行 1.0.5 安装包时会自动识别 1.0.4 的原安装目录，无需重新选择位置。
- 覆盖安装会保留收藏、设置、UV 元数据和缓存；安装目录旁的 `FBX_Data` 会在更新期间临时保护并原位恢复。
- 便携版不自动覆盖自身，仍提供发布页作为手动下载入口。
- 左侧列表中的 FBX 模型可以直接拖到 3ds Max 等支持文件拖放的外部程序。
- 缩略图视图中的模型同样支持向外拖放。
- 临时拖入查看器的 FBX 也可以再次作为原始文件拖出。
- 拖放图标调整为 32×32，减少对 3ds Max 视口的遮挡。
- 右键模型可以复制同名 `.mesh` 路径；程序会优先检查同目录，再逐级检查上级目录。
- UV 参数拖动更加细腻，支持 `Shift` 精调、`Ctrl` 加速和双击单项重置。
- UV 参数支持 `Ctrl+Z` 撤销以及 `Ctrl+Y` / `Ctrl+Shift+Z` 重做，平铺值可以正确回到 `0`。
- 修复模型加载后不可见、鼠标视角控制无响应，直到手动调整 UV 参数才刷新画面的问题。
- 新增顶部 `?` 帮助入口，按 `F1` 可打开中文操作速查。
- 模型列表和缩略图悬停时显示小型拖拽提示与“更多操作”按钮，不再依赖用户自行发现右键功能。
- 当前模型路径栏新增显式 `.mesh` 复制按钮，复制成功或查找失败会显示短提示。
- 首次加载目录和首次启用贴图预览时显示一次性操作提示，关闭后会记住状态。
- 首次启动可选择是否显示 `.mesh` 操作、默认视口操作模式、列表或缩略图视图，以及是否打开操作帮助；之后可通过目录栏齿轮按钮重新调整。
- 缩略图收藏恢复到右上角，“⋯”移动到左下角；当前路径的 `.mesh` 操作改为紧凑的 `M` 按钮。
- 搜索框移动到排序和筛选工具栏下方，减少长列表中的鼠标移动距离。
- 3ds Max 操作模式支持在中键拖动过程中按下或松开 `Alt`，即时切换平移和旋转。
- 手动检查到新版本时会直接显示 GitHub Release 更新说明，并可选择打开发布页。
- U/V 标签显示轻量 `↔` 拖动提示；单项数值偏离默认值时显示 `↶` 重置按钮，点击后恢复默认并自动隐藏。
- 搜索框聚焦样式更加克制，点击画布或搜索区外区域会立即取消聚焦。

## 安全与稳定性

- 拖出请求只接受当前应用窗口发起的消息。
- 只允许拖出绝对路径、真实存在的 `.fbx` 文件。
- 渲染层使用专用 preload API，不再通过通用 IPC 发送接口启动文件拖放。
- Electron 升级到 43.1.0，electron-builder 升级到 26.15.3。
- `npm audit` 已无已知依赖漏洞。

## 使用说明

在列表或缩略图中按住模型并拖到 3ds Max 视口。Max 收到的是原始 FBX 文件，后续是否显示 FBX Import 对话框由 Max 的导入设置决定。

需要在编辑器中引用 `.mesh` 时，右键模型并选择“复制 .mesh 路径”。例如 `xxx/fbx/model.fbx` 可以匹配到 `xxx/model.mesh`。

模型悬停时也可以点击“⋯”打开相同操作；当前模型可直接点击路径栏右侧的 `M` 按钮复制 `.mesh` 路径。按 `F1` 可随时查看完整操作速查。

调整贴图预览参数时，可以拖动 U/V 标签；按住 `Shift` 进行精调，按住 `Ctrl` 加速，双击标签恢复该项默认值。

如果 3ds Max 以管理员身份运行而本程序没有提权，Windows 可能阻止拖放。此时请让两个程序使用相同权限级别运行。

本版本支持 Windows 10 / 11。

从源码开发或打包需要 Node.js 22.12 及以上；直接运行安装版或便携版不需要安装 Node.js。

自动更新从安装此版 1.0.5 后开始生效。旧版 1.0.4 仍需手动安装一次新版，之后安装版可自动下载后续版本。
