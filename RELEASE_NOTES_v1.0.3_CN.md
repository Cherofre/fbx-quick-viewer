# FBX 快速预览器 v1.0.3 发布说明

## 新增

- 支持将单个 `.fbx` 文件直接拖入预览区进行临时预览。
- 拖入 FBX 后，左侧顶部会显示临时项，同时保留已打开的文件夹列表。
- 大目录缩略图改为按滚动位置懒加载，避免一次性处理几千上万个模型导致卡顿。
- 超大列表仍提供“生成前 300”和“生成全部”的手动入口。
- 新增拖拽预览与缩略图懒加载的最小回归测试。

## 修复与优化

- 修复拖入 FBX 被贴图拖拽逻辑拦截的问题。
- 修复临时拖入 FBX 在键盘/索引选择时可能拼出错误路径的问题。
- 修复缩略图异步生成期间列表重渲染后可能写错目标格子的问题。
- 大列表懒加载缩略图前会优先读取磁盘缓存，减少重复生成。
- 避免手动批量缩略图任务与正在运行的懒加载任务并发。

## 后续规划

- 已记录多 UV / UV2 展开图与贴图通道预览功能规划。
